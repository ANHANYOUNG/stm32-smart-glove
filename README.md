# Smart Glove

STM32-based smart glove firmware for recognizing sign gestures and converting them to text.

## Files
- `Core/`, `Drivers/` : firmware source files
- `project.ioc` : STM32CubeMX configuration

## Environment
- MCU: STM32F103RBT6
- IDE: STM32CubeIDE
- Language: C with HAL

## Build
1. Open the project in STM32CubeIDE.
2. Build the project.
3. Flash it to the board with ST-LINK.
